## Critical Technical Notes

### PDF Parsing (DO NOT BREAK)
Chase changed their Zelle PDF to a table layout. Sender name sits between
status keyword and the `$` amount sign. The sender regex MUST use `$` as
the terminator (NOT "In moments"):

```javascript
const senderMatch =
  fullBlock.match(/(?:Completed|Delivered)\s+"[^"]*"\s+([A-Z][A-Za-z\s\-\.]+?)\s+\$/) ||
  fullBlock.match(/(?:Completed|Delivered)\s+([A-Z][A-Za-z\s\-\.]+?)\s+\$/);
```

### Name Matching Config

const DATE_TOLERANCE_DAYS = 2;   // CSV posts 1 day after PDF date
